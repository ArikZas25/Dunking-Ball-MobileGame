# 🏀 Dunking Ball

> A hyper-casual mobile game where you control a ball and shoot it through hoops — built with Unity.

---

## 🎮 About the Game

Dunking Ball is a fast, fun, and addictive hyper-casual mobile game. The goal is simple — control your ball and get it through as many hoops as you can. Easy to pick up, hard to put down.

The game was built from scratch using the **Unity game engine**, with custom shaders and C# gameplay logic.

---

## 📱 How to Play

- **Tap / swipe** to aim and launch your ball
- Guide the ball through the hoops
- Chain shots to build your streak
- Don't miss — keep the run going!



---

## 🛠️ Built With

- **Unity** — game engine
- **C#** — gameplay scripting
- **ShaderLab / HLSL** — custom visual shaders for a polished look

---

## 🚀 Running the Project

1. Clone the repo
2. Open the project in **Unity** (recommended: Unity 2021 LTS or later)
3. Hit **Play** in the editor or build to Android / iOS

```bash
git clone https://github.com/ArikZas25/Dunking-Ball-MobileGame.git
```
